## 목표
채팅 입력창 하단에 모델 선택 드롭다운을 추가한다.
표시명(label)과 API 전송값(value)을 분리하여 관리한다.

## 데이터 설계 결정
`{ label: string; value: string; provider: string }` 배열 + `selectedModel` state(value 저장)
- label: 사용자에게 표시 ("claude-sonnet-4-6")
- value: API 전송값 ("anthropic/claude-sonnet-4-6")
- 전송 시 변환 불필요 — selectedModel을 그대로 body에 담으면 됨
