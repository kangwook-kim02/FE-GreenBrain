## 관련 파일
- `app/chat/page.tsx` — 핵심 변경 파일

## 작업 분해
- [ ] `currentSessionId` 상태 추가 (첫 메시지 전 null)
- [ ] `username` → `user?.nickname ?? '환경지킴이'` 교체
- [ ] `Message.carbonCost` 타입을 `number | null | undefined`로 확장
- [ ] setTimeout mock 제거 → async handleSubmit 구현
  - 세션 없으면 `POST /api/chat/sessions` 호출 후 id 저장
  - `POST /api/chat/sessions/{id}/messages` 호출
  - 응답 `tokens_remaining` → `updateRemainingTokens`
  - 응답 `exhausted: true` → tokens 0으로 처리됨 (자동 소진 화면)
  - 응답 `carbon_gco2eq: null` → CarbonCard "탄소 계산 불가" (이미 지원)
- [ ] 에러 처리: 403 → `updateRemainingTokens(0)`, 502 → 에러 메시지 말풍선
- [ ] CarbonCard analogy: carbonCost null일 때 undefined 전달
