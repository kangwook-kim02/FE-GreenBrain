## 목표

디자인 프로토타입 기반으로 `/onboarding` 페이지 3단계 스텝 UI 구현.
교통수단 → 식단 → 주거 형태 선택 카드 + 프로그레스바 + 이전/다음 버튼.

## 관련 파일

- `app/onboarding/page.tsx` — 현재 placeholder → 전체 구현
- `DesignPrototype_GreenBrain/src/app/pages/Onboarding.tsx` — 디자인 프로토타입 참조
- `contexts/AppContext.tsx` — user.onboarding_complete 상태 참조

## 작업 분해

- [ ] 프로그레스바 (1/3, 2/3, 3/3) + 단계 표시
- [ ] 1단계: 교통수단 선택 카드 (대중교통/자가용/자전거/도보) — 인라인 SVG 아이콘 + 라벨
- [ ] 2단계: 식단 선택 카드 (비건/채식 위주/균형 식단/육식 위주)
- [ ] 3단계: 주거 형태 카드 (아파트/단독주택/원룸·오피스텔/셰어하우스)
- [ ] 선택된 카드: 초록 테두리 + 배경 + 체크 아이콘 강조
- [ ] 이전/다음 버튼 (선택 전 다음 비활성화)
- [ ] 반응형 대응 (375px ~ 1280px)
- [ ] Next.js App Router 적용 (useNavigate → useRouter, 'use client')

## 완료 기준

- 3단계 스텝 모두 정상 동작
- 선택 카드 초록 테두리 + 체크 아이콘 표시
- 선택 없이 다음 버튼 비활성화
- 3단계 완료 후 /chat으로 이동
- 모바일(375px) ~ 데스크탑(1280px) 반응형
- emoji 대신 인라인 SVG 사용
- React.FC 미사용, alert() 미사용, Tailwind 클래스만 사용
