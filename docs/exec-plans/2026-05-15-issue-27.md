## 주요 변경 사항 (GreenBrain-API.md 기준)

### 챌린지
| 구분 | 기존(placeholder) | 확정 |
|------|------------------|------|
| Challenge 타입 | `id, title, description, category(한국어), icon` | `id, category(transport/diet/energy), title, description, difficulty(1~3), status, created_at, completed_at` |
| 챌린지 조회 | `GET /api/challenges/current` → `{ challenge, daily_count }` | `GET /api/challenges/current` → `{ challenge: Challenge \| null }` |
| 챌린지 생성 | `POST /api/challenges/{id}/accept` | `POST /api/challenges/generate` → `{ challenge, created }` |
| 챌린지 수락 | (없음) | `POST /api/challenges/{id}/accept` → `{ challenge }` |
| 사진 업로드 | `POST /api/challenges/{id}/verify` → `{ tokens_remaining }` | `POST /api/challenges/{id}/photo` (FormData) → `{ photo, challenge, reward }` |

### 피드
| 구분 | 기존(placeholder) | 확정 |
|------|------------------|------|
| FeedItem 타입 | `id, userId, username, challengeTitle, imageUrl, likes, likedByMe, isOwner, timestamp` | `photo_id, challenge_id, user_id, nickname, profile_image_url, title, category, photo_url, like_count, liked_by_me, carbon_saved_gco2eq, created_at` |
| 피드 조회 | `GET /api/feed` → `{ items, has_next }` | `GET /api/challenges/feed?limit&offset` → `{ items, total, limit, offset }` |
| 피드 삭제 | `DELETE /api/feed/{item_id}` → 204 | `DELETE /api/challenge-photos/{photo_id}` → `{ deleted, photo_id, deleted_at }` |

### 좋아요
| 구분 | 기존(placeholder) | 확정 |
|------|------------------|------|
| 좋아요 토글 | `POST /api/feed/{item_id}/like` → `{ likes, liked_by_me }` | `POST /api/challenge-photos/{photo_id}/like` → `{ liked, photo_id, like_count, reward_given, reward_amount, tokens_remaining }` |
| 좋아요 목록 | (없음) | `GET /api/challenge-photos/{photo_id}/likes?limit&offset` → `{ items, total, limit, offset }` |
